Cliente MySQL Web. PHPmyadmin

Como para instalar la versión Servidor hemos instalado repositorios variados, ya podemos instalar php ya que se encuentra en la lista de uno de ellos con: sudo yum install php



Ahora reiniciamos apache: sudo service httpd restart


Instalamos el repositorio 7-5


Para borrar y corregir todos los errores que podemos tener en nuestro sistema por acumulación de repositorios mal instalados y otros


Instalamos ahora phpmyadmin sudo yum install phpmyadmin


Ahora desde un cliente escribiendo la dirección de nuestro servidor seguido de /phpmyadmin/ entramos en el servidor web de MySQL
