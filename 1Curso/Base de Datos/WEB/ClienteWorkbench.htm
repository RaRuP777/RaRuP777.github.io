Cliente MySQL Escritorio. Workbech

MySQL escucha pediticiones desde el puerto 3306, pero el Firewall nos impide el acceso. Para modificar las configuraciones se hace a través de IPTables. Esto nos hace falta para conectar un cliente de escritorio a la base de datos de MySQL.

Para conectar un cliente web necesitamos tener instalado en el server un servidor web, en este caso nos hace falta phpmyadmin para poder desde el navegador acceder. Para esto también nos hace falta el lenguaje de programación php.

Desde Workbench

Mediante SSH, abrir Workbench crear nueva conexión usando estandar tcp/ip via SSH (como esta abierto un canal en el firewall el acceso esta permitido por el puerto 22) sshhostname ip:puerto, (192.168.12.134:22) user name root mysql hostname 127.0.0.1 mysql server port 3306 username root.


Para permitir la conexiones al puerto 3306 de MySQL que es donde MySQL-Server escucha las peticiones:

sudo iptables -A INPUT -i eth0 -0 tcp dport 3306 -j ACCEPT

service iptable save

y luego reiniciamos el servicio

sudo mysqld restart
