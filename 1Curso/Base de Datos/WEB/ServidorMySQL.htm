Servidor MySQL-Server. Instalación y Configuración en CentOS

Se ejecuta putty para conectarse remotamente a la máquina virtual de CentOS y se loguea con los datos de usuario “root” y contraseña.

Se nos recomienda cambiar la contraseña Root...

Linux funciona a base de paquetes. Para obtener un programa se necesitan los paquetes que lo forman. Si no existen habrá que buscarlos. Para facilitar su búsqueda se utilizan los repositorios, donde se almacenan las direcciones de los paquetes a descargar.

En este caso, CentOS necesita actualizar los repositorios. Así que conectándose a Internet desde otro equipo y se buscan los archivos de repositorios y se descargan.

Se transfieren los archivos necesarios para descargar los repositorios mediante WinSCP a la localización adecuada dentro de la máquina virtual de CentOS (/etc/yum.repos.d/)

 

En CentOS, situándose en la carpeta adecuada (/etc/yu.repos.d), se instalan los archivos de los repositorios, que son dos. El primero mediante la orden “sudo yum install mysql-community-release-el17-5.noarch.rpm”.

Y el segundo con “sudo yum rpm –ivh epel-release-6-8.noarch.rpm”.

Como el servidor CentOS no tiene DNS se actualiza el archivo de DNS (/etc/resolv.conf) a través de WinSCP añadiendo la dirección de DNS, nameserver 8.8.8.8 y nameserver 8.8.4.4 (dirección del DNS de Google).

 

Ahora sí se puede instalar MySQL porque los repositorios ya encuentran los paquetes necesarios. Mediante “sudo yum install mysql-server mysql php-mysql” se inicia la instalación.

Terminada la instalación se inicia el servicio MySQL con “sudo service mysql start”.

            Y finalmente se puede usar MySQL en este máquina usando la orden “mysql” desde el terminal.
