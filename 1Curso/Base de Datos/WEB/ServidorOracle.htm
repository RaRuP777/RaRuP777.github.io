Instalación y Configuración de Servidor Oracle en CentOS

En lugar de instalarlo desde un repositorio. Para instalar Oracle vamos a la página web oficial de Oracle y descargamos el paquete de instalación.

Una vez lo tenemos en nuestro equipo local lo pasamos via WinSCP al servidor CentOS

Despues mediante rpm ivh Oracle-xe-11.2.0-1.0.x86_64.rpm instalamos este paquete en nuestro servidor CentOS

Para que funcione todo correcto también instalamos unos paquetes adicionales que nos pueden hacer falta para el buen funcionamiento del programa como son libaio bc flex

Ahora nos queda configurar el servidor, vamos a la ruta /etc/init.d/ y allí ejecutamos ./oracle-xe configure primero definimos el puerto para HTTP al 8090 ya que el 8080 está siendo utilizado ya

Y ahora especificamos el puerto de escucha de la base de datos

También elegimos que el servicio Oracle se inicie cuando se inicia el servidor

Lanzamos el script /u01/app/oracle/product/11.2.0/xe/bin/oracle_env.sh con source ./oracle_env.sh para incluir en el #path (cuando ejecutemos oracle no tendremos que estar en la misma carpeta que el archivo ejecutable)

Y ya está funcional y preparada para ser usado servidor Oracle de base de datos
